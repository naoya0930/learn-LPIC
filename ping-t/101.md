# LDIC
### ハードウェア設定の決定と設定分野
- BIOSとUEFIの違い
  - BIOS...古い
    - 基本デバイスのチェックと初期化
    - OSが起動する前に実行する
  - UEFI...新しい
    - セキュアブートが可能
### デバイス情報の確認
- `/proc/cpuinfo`
  - CPU情報
- `/proc/interrupts`
  - IRQ情報
  - Interrupt Request
- `/proc/ioport`
  - i/Oアドレス
- `/proc/meminfo`
  - メモリ情報
- `/proc/bus/usb/*`
  - USBデバイス情報
- `/proc/bus/pci/*`
  - PCIデバイス情報

# ファイル共有サーバ

## Samba

### Windows基礎知識
#### ワークグループ構成
- 管理用のサーバをもたず、各サーバが独立して存在
- 特定のサーバがマスターブラウザとして稼働する
- 管理認証権があるわけではない

#### NTドメイン構成
- windows端末の集中管理を実施
- Windows NT, 9X, 2000以前の構成
- NTLM(NT LAN Manager)というチャレンジ・レスポンス方式で実施
  - LDAP,Kerberosに非対応だった
- 管理系統は以下
  - PDC(Primary Domain Controller)
    - ユーザ作成・削除・パスワード変更
    - 唯一の書き込み可能ノード
  - BOC(Backup Domain Controller)
    - PDCのレプリケーション
    - 認証要求を処理
    - SAMは読み取り専用
    - PDC障害時に昇格可能
  - 他PC

#### AD構成
- Active Directory構成
- ドメインコントローラ(DC)
  - ADを提供するサーバを保有
- ドメインに所属する形でPCが存在
  - DCはレプリケーション構成を想定している
